Description:
ARSC_CHK is a low-level tool for checking ARSC library functions that
perform soundcard input & output. Use arsc_chk to
(1) check volts-full-scale of the soundcard A/D, D/A converters,
(2) check soundcard output,
(3) check soundcard loopback latency, and
(4) create a registry file with ARSC parameter values.

Installation: 

(1) Run Setup.

(2) ARSC_CHK requires Microsoft C-runtime libraries. If Windows complains that
'This application has failed to start because the application configuration
is incorrect.', download the C-runtime installation program from
http://audres.org/downloads/vcredist_x86.exe.

(3) To remove existing ARSC entries in the Windows registry double-click
on c:\Program Files\BTNRH\arsc\no_arsc.reg.
